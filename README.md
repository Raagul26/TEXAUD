# TEXAUD
Simple offline Text to Audio converter in python. Created using pyttsx3 module.

## Recommended

#### Command:  

    >>pip install -r requirements.txt 
    >>py Texaud.py
         or
    >>python3 Texaud.py

#### Supporting platform:
     Windows
     Linux
     Mac os
